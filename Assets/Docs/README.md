# Documentación

Aquí encontraremos toda la documentación del proyecto, incluido el High Concept,
el Level Design, y definiciones. También encontraremos documentación pertinente
al código.

# Estructura de carpetas

- `Assets/Art/Materials/`: todos los materiales propios a utilizar.
- `Assets/Art/Models/`: modelos.
- `Assets/Art/Textures/`: texturas.
- `Assets/Audio/Music/`: música (interactiva o no).
- `Assets/Audio/Sound/`: sonidos interactivos.
- `Assets/Code/Scripts`: C# scripts.
- `Assets/Code/Shaders`: Shaders.
- `Assets/Docs/`: lo que vemos aquí.
- `Assets/Level/**/*`: todo aquello relacionado con el game design.
